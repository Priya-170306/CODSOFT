# SecurePass 🔐

SecurePass is a simple password generator built using Python and CustomTkinter.
It allows users to quickly create passwords with different strength levels and choose the desired length.

## Features ✨ 

* Generate passwords with Easy, Medium, and Strong levels
* Adjustable password length using a slider
* Dark and Light mode switch
* Clean and simple interface
* Double-click to copy the generated password

## How the Program Works  📌 

The application takes a few inputs from the user such as name, favorite word, and birth year.
Based on the selected strength level, the program creates a password using different logic.
For strong passwords, random characters, numbers, and symbols are used.

## Technology Used  🛠️ 

* Python
* CustomTkinter (GUI library)

## Running the Project ▶️ 

1. Install the required library
   pip install customtkinter

2. Run the program
   python main.py

## Project File 📄

main.py – main application file containing the password generator code.

## Purpose 🎯

This project was created to practice Python GUI development and basic password generation logic.
