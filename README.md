# 🧮 Smart Calculator

## 📌 About Project
This is a calculator application that I created using Python and Tkinter as part of my internship task. It can perform basic arithmetic operations and has a simple and clean user interface.

## 🚀 Features
- Supports addition, subtraction, multiplication, and division
- Includes percentage calculation
- Clear (C) button to reset input
- Backspace (⌫) to remove last value
- Handles invalid input errors

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)

## ▶️ How to Run
1. Make sure Python is installed
2. Run the file using:
   python calculator.py

## 💡 What I Learned
- Basics of GUI development using Tkinter
- Handling user input and events
- Improving UI layout using grid system
